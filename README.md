## Getting Started

Create a new Astro project with Tailwind CSS:

```sh
pnpm create astro@latest -- --template with-tailwindcss
```

Or clone this repository and install dependencies:

```sh
git clone https://github.com/DylanZM/Astro-app.git
cd Astro-app
pnpm install
pnpm run dev
```
